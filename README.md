## What is this?

A multi-agent deliberation system where 8 AI personas Рђћ built from ~3,500 tokens of researched personality data each Рђћ argue existential questions using **any free OpenAI-compatible LLM**.

No paid API keys. No cloud dependencies. Just pure intellectual chaos.

## Features

- **K-ZERO Console** Рђћ Interactive god-mode. Inject variables mid-debate. Whisper to individual agents. Watch consequences ripple through the group.
- **8 deeply researched personas** Рђћ Not "be creative" prompts. Real cognitive biases, speech patterns, memory seeds, and clash dynamics sourced from primary texts.
- **Any free LLM API** Рђћ Groq, OpenRouter, Google AI Studio. Pick one, plug it in, run.
- **Post-simulation analysis** Рђћ LLM-powered transcript analysis extracts agreement maps, position tracking, emergent insights, and topic clusters.
- **Interactive Dash dashboard** Рђћ Network graphs, conflict heatmaps, position timelines, and key quotes. All in Plotly.
- **3 pre-built scenarios** Рђћ Meaning of life, AI alignment, death and legacy. Or write your own.
- **Shareable transcripts** Рђћ Every session saves to JSON. Replay, analyze, share.

**1. Clone**

```bash
git clone https://github.com/ksk5429/kzero.git

## K-ZERO Console

You are the demiurge. The council exists because you willed it.

```
K-ZERO > /all What is worth dying for?
```

Watch 8 agents respond in character, building on each other's arguments.

```

         will remember any of you in 200 years.

Watch the shockwave. Musk doubles down. Sartre shrugs. Kobe doesn't care. Carlin saw it coming.

K-ZERO > /ask feynman Does Bryan's immortality quest contradict everything 
         you believe about curiosity?

Pull one thread. Watch the whole tapestry shift.

### All Commands

| Command | What it does |
|---------|-------------|
| `/all <question>` | Pose a question Рђћ all 8 agents respond |
| `/ask <name> <text>` | Whisper to one agent directly |
| `/inject <text>` | God-mode variable injection Рђћ mid-debate plot twist |
| `/next` | Let the next round play out naturally |
| `/next <N>` | Run N rounds without intervention |
| `/who` | Show all members and their current stance |
| `/history` | Show recent transcript |
| `/synthesis` | Kevin synthesizes the discussion so far |
| `/end` | End session and save transcript |

## Add Your Own Council

Each character is a folder under `characters/` with 5 files:

1. **`personality_matrix.json`** Рђћ Cognitive biases, knowledge domains, interaction weights
2. **`voice.md`** Рђћ How they talk. Sentence structure, vocabulary, verbal tics
3. **`axioms.md`** Рђћ 10 non-negotiable beliefs
4. **`clash_points.md`** Рђћ How they react to every other member
5. **`memory_seeds.md`** Рђћ Formative experiences the LLM can reference

Use primary sources. Biographies, interviews, their own writing. The richer the data, the more the agent sounds like *them* instead of a chatbot cosplaying.
